# MANTRAS — Landing Page Experience

Cinematic conversion-focused landing page created for a digital product in the spirituality/self-development niche.

Built with a strong focus on:

- premium UI
- immersive storytelling
- smooth animations
- responsive experience
- modern dark aesthetic

---

## Stack

- React 19
- Vite
- TypeScript
- TailwindCSS v4
- Framer Motion
- Lucide React

---

## Features

- Fully responsive layout
- Cinematic hero section
- Scroll-based animations
- Reusable component architecture
- Premium dark/gold design system
- Optimized for GitHub Pages deployment
- Mobile-first responsive behavior
- SEO-ready structure
- Smooth hover + glow interactions

---

## Purpose

This project was built to create a modern high-converting landing page experience for a premium digital product.

The main goal was combining:

- cinematic visual direction
- luxury branding
- immersive frontend interactions
- conversion-focused structure

into a fast and polished React application.
